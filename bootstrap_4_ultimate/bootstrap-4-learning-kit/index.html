Bootstrap 4 Learning Kit

Welcome to this custom made Bootstrap 4 Learning Kit.

This learning kit consists of the following two sections:

  • Reference
  • Practice

Choose a topic from the dropdown menu. You'll then go to that topic's reference page. From there you can toggle between the two different views: reference and practice.

At anytime you can go to another topic from the dropdown menu.
